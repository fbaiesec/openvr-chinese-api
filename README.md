# openvr-chinese-api
ValveSoftware公司openvr接口api的中文翻译
OpenVR可以让VR应用开发者忽略不同VR硬件平台的细节，提供了一个统一的访问各种VR硬件平台的运行时。  
这个代码库提供了原英文版接口说明的中文翻译。运行时就是Steam平台下的工具SteamVR。 

[英文原版wiki](https://github.com/ValveSoftware/openvr/wiki/)
